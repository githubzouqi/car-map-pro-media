# Android 版现场运维工具定制开发


## Media 版本简述

**隐藏小车的路径实时显示**、
**不显示工作站**、
**安全区域的建立以及解除功能**、
**单点安全区域的建立以及解除功能**、
**除上面说的一些功能外，更多功能请看** ： *Media 定制开放功能部分*

## Media 定制开放功能 

* 1、地图说明
* 2、地图信息
* 3、地图复位
* 4、小车电量
* 5、按钮支持地图的放大和缩小、屏幕触摸支持地图的放大和缩小
* 6、其他：查看某小车的状态
* 7、其他：查看pod信息
* 8、其他：重发任务
* 9、其他：下线某小车
* 10、其他：驱动pod去目标点位（可用存储区）
* 11、其他：更新pod地址
* 12、其他：释放pod状态
* 13、其他：更细地址状态（将地址状态改为status：Available以及lockby：0）
* 14、其他：小车上移一格
* 15、其他：小车下移一格
* 16、其他：小车左移一格
* 17、其他：小车右移一格
* 18、其他：清除小车所有路径
